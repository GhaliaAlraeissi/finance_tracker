Description:
This program is a Personal Finance Tracker that allows users to manage their income and expenses. Users can add income, record expenses with categories, and view a financial summary showing total income, total expenses, and savings. The program also includes features to sort expenses and display spending trends using a bar chart. Data is saved to a file so it persists between sessions.

Libraries Used:

* json
* matplotlib

Installation Instructions:
Install required libraries using pip:
pip install matplotlib
